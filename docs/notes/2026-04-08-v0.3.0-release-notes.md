# SheetForge MCP v0.3.0 Notes

Date: 2026-04-08
Status: Released

## What Shipped

Version `0.3.0` introduced three major improvements:

1. Uniform JSON response envelopes for the public MCP tool surface.
2. `dry_run` previews for core write, formatting, and range mutation operations.
3. New table-oriented tools:
   - `append_table_rows`
   - `update_rows_by_key`

## Public Response Contract

All tools now return a predictable top-level JSON shape:

```json
{
  "ok": true,
  "operation": "tool_name",
  "message": "Human-readable summary",
  "data": {}
}
```

Error shape:

```json
{
  "ok": false,
  "operation": "tool_name",
  "error": {
    "type": "DataError",
    "message": "What went wrong"
  }
}
```

For preview-capable destructive tools, responses may also include:
- `dry_run`
- `changes`
